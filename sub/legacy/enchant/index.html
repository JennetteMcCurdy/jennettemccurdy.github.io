Anvil Prices Explained

as of 1.17.1

Anvils are mainly used for repairing items, combining enchantments, and giving items a custom name. All of these activities cost experience levels from the player. Anvil uses are the number of times an item has been used in an anvil. The more anvil uses an item has, the more expensive it will be to use in an anvil once again. If the use price calculated is to be above 39 levels, the anvil will reject the item with the error "Too Expensive!"

This is the formula Minecraft uses for calculating an anvil price: 2^(anvil uses)-1 = Price
